# SkinLesionClassification_HAM10000
This is for CSE404 Group Project
- We worked on some baseline models that are showed in LogRegRanFor.ipynb.
- A simple CNN architecture was used, shown in NeuralNetBaseline.ipynb
- Our best and recommended model is in ResNet50_TransferLearning.ipynb which has the best accuracy and melanoma recall. The trained model is stored in resnet50_ham10000.pth file. Use any library documentation to load that file into your model.
- EfficientNet.ipynb is an exploratory file which ahsn't been documented well enough right now.
- Note: Each file uses its own way of loading the data so be mindful about that.
